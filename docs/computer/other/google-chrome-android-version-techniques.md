# Google Chrome (Android 版) 使用技巧

以下適用於 Android 版 Google Chrome。

## 禁用 Tab Groups

Android 版的 Google Chrome 原本分頁是呈現垂直連續排列，更新後變成格狀排列，非常難用。

設定 Chrome Flags 的 Tab Groups 為 Disabled 後重新啟動 Chrome。

> Tab Groups
>
> chrome://flags/#enable-tab-groups
>
> Allow users to create groups to better organize their tabs. – Android
