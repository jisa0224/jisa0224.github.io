# 開源人年會 COSCUP 2018 會後筆記(未完成)

我今年參加了「[COSCUP x GNOME.Asia x openSUSE.Asia 2018](https://2018.coscup.org/)」這個活動，它會舉辦很多演講，然後你可以自己挑自己想聽的活動去聽。

我挑幾個我聽完覺得重要的寫一下，按照類別來分

## 社群類活動

  - [Why should you get involved in open source?](https://2018.coscup.org/programs/why-get-involved/)
  - [我在 GitHub 參與開源貢獻專案的日子](https://2018.coscup.org/programs/github/)
  - [My openSUSE Journey](https://2018.coscup.org/programs/my-opensuse-journey/)
  - [開源軟體與社群 - 參與國際社群經驗談](https://2018.coscup.org/programs/global-community/)
  - [Building popular open source projects](https://2018.coscup.org/programs/building-popular-open-source-projects/)

前四個是我本次最重要的內容，因為我想參與開源卻不知道如何入門，第五個則是好的專案要有的一些要素。

前四個裡面有提到一些資源

  - [Google Summer of Code](https://summerofcode.withgoogle.com/)
  - [openSUSE 101](https://101.opensuse.org/)
  - [Hacktoberfest](https://hacktoberfest.digitalocean.com/)
  - [24 Pull Requests](https://24pullrequests.com/)
  - [OpenHatch](https://openhatch.org/)
  - [GNOME Travel sponsorship](https://wiki.gnome.org/Travel)
  - [openSUSE Travel Support Program](https://en.opensuse.org/openSUSE:Travel_Support_Program)
