# Microsoft Office Word 修正變更字體大小後行距跑掉的問題

1. 相信各位都遇過這樣的問題：改了字型的大小後，行距放大的超乎想像，例如將Word字型由12點改成14點，行距被拉大的有些不成比例，原因在哪裡呢？  
   ![](microsoft-office-word-font-size-and-line-spacing/0.png)  
   ![](microsoft-office-word-font-size-and-line-spacing/1.png)

2. 請各位開啟格線顯示\[檢視\>顯示\>格線\]  
   ![](microsoft-office-word-font-size-and-line-spacing/2.png)

3. 執行\[常用\>段落\>顯示\[段落\]對話方塊\]，發現Word預設會勾選\[文件格線被設定時，貼齊格線\]，其實格線在排版對位時，是一個很好的輔助工具，但若只是一般的文件編輯，將它取消，這樣在設定行距時，Word就會乖乖的聽話...  
   ![](microsoft-office-word-font-size-and-line-spacing/3.png)

4. 取消選\[文件格線被設定時，貼齊格線\]，行距就恢復正常  
   ![](microsoft-office-word-font-size-and-line-spacing/4.png)
