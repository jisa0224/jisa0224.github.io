# Bash 如何展開 * 自動補全

如果某個資料夾下有 a、b、c 三個檔案，然後下

```
$ ls *
```

此時在 `*` 後立刻按下 `Ctrl-X *`，就會展開成

```
$ ls a b c
```

在 Bash 中此功能稱為 `glob-expand-word`，`man bash` 中寫

> The word before point is treated as a pattern for pathname  
> expansion, and the list of matching filenames is inserted,  
> replacing the word. If a numeric argument is supplied, an  
> asterisk is appended before pathname expansion.

不過並非每一個 Bash 的設定都是這組組合鍵，如果要查看當前設定的話，可以下

``` shell
bind -p | grep glob-expand-word
```

另外還有一個功能 `glob-list-expansions`，可以顯示出展開後的內容，但不替換進當前命令行，預設組合鍵是 `Ctrl-X g`

> The list of expansions that would have been generated by  
> glob-expand-word is displayed, and the line is redrawn. If a  
> numeric argument is supplied, an asterisk is appended before  
> pathname expansion.
